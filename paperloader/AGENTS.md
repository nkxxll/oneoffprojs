# Project Overview

**paperloader** is a Python project using `uv` for dependency management.

## Setup

```bash
uv sync
```

## Run

```bash
uv run main.py
```

## Test

No tests configured yet. To add tests:

1. Install test framework: `uv add --dev pytest`
2. Create test files matching `test_*.py` or `*_test.py`
3. Run: `uv run pytest`

## Key Files

- `main.py` - Entry point
- `pyproject.toml` - Project config (Python 3.14+, no current dependencies)
